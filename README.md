#Tic Tac Toe in Elm


elm-package install
npm install

To Run Program Locally:
    ```npm start```
then navigate to ```localhost:8080```

Run tests once:
    npm test

Run integration tests
In one tab:
    java -jar selenium-server-standalone-2.53.1.jar wdio.conf.js

In another:
    npm run

And in one more...
    npm run itest
